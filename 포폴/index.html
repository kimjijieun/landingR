TO STUDY

  • Gsap

    GSAP를 활용하면 화려하고 인터랙티브한 화면을 쉽고 빠르게 제작할 수 있습니다. 현재는 간단한 애니메이션 효과만 적용해 보았지만, GSAP의 다양한 기능을 활용하면 복잡한 애니메이션도 간결하고 효율적으로 구현할 수 있다는 점을 알게 되었습니다. 앞으로 이를 익히고 프로젝트에 적극적으로 활용할 수 있도록 꾸준히 학습할 계획입니다.

  • Vanilla JS

    React, Vue.js와 같은 JavaScript 프레임워크 및 라이브러리의 기초를 이해하기 위해 현재 Udemy에서 바닐라 JavaScript 강의를 구매하였고 수강하고자합니다. 제이쿼리를 이용해 작업한 프로젝트들을 JavaScript로 똑같이 구현하는 것을 목표로 삼고 있습니다.

  • Three.js & Canvas

    인터랙티브한 화면을 위해 Three.js 강의를 통해 웹 브라우저에서 3D 그래픽을 렌더링하기 위해 사용되는 JavaScript 3D 라이브러리 Three.js를 학습하고 있습니다.

  • REACT

    React는 프론트엔드 개발자로 성장하기 위해 꼭 필요한 기술이라고 생각합니다. React를 배워 현재의 웹 개발 스킬을 더욱 향상시키고, 지금보다 더 다양한 프로젝트를 경험하며 실력을 쌓아가고 싶습니다. 이를 통해 동적이고 사용자 친화적인 웹 애플리케이션을 구현할 수 있는 개발자가 되겠습니다.

  • TypeScript

    React의 프로젝트를 더욱 안정적이고 가독성 높게 만들 수 있는 TypeScript를 같이 학습하려고 합니다. TypeScript의 코드 안정성, 가독성 향상, 자동 완성, 코드 품질 향상 등의 다양한 장점들을 활용하여 높은 퀄리티의 React 프로젝트를 만들어가는데 기여하겠습니다.

JavaScript 프레임워크, 라이브러리, CSS4와 같은 새로운 웹 기술들을 빠르게 학습하고 사용자 경험에 대한 최신 트렌드를 한발 앞서서 받아들이는 퍼블리셔가 되고자 합니다.
