TO STUDY

  • Vanilla JS

    React, Vue.js와 같은 JavaScript 프레임워크 및 라이브러리의 기초를 이해하기 위해 현재 Udemy에서 바닐라 JavaScript 강의를 구매하여 수강하고 있습니다. GSAP의 다양한 애니메이션들을 JavaScript로 똑같이 구현하는 것을 현재의 목표로 삼고 있습니다.

  • Three.js & Canvas

    인터랙티브한 화면을 위해 Three.js 강의를 통해 웹 브라우저에서 3D 그래픽을 렌더링하기 위해 사용되는 JavaScript 3D 라이브러리 Three.js를 학습하고 있습니다.

  • REACT

    리액트는 어쩌고저쩌고 프론트엔트로써 어쩌고저쩌고 필요한 기술이기에 어쩌고저쩌고 공부하겠습니다. React를 배워 현재의 웹 개발 스킬을 더욱 향상시키고 지금보다 더 다양한 프로젝트를 접할 기회를 가지고 싶습니다.

  • TypeScript

    React의 프로젝트를 더욱 안정적이고 가독성 높게 만들 수 있는 TypeScript를 같이 학습하려고 합니다. TypeScript의 코드 안정성, 가독성 향상, 자동 완성, 코드 품질 향상 등의 다양한 장점들을 활용하여 높은 퀄리티의 React 프로젝트를 만들어가는데 기여하겠습니다.

JavaScript 프레임워크, 라이브러리, CSS4와 같은 새로운 웹 개발 툴들은 끊임없이 등장하고 발전하고 있습니다. 빠르게 변화하는 웹 기술과 사용자 경험에 대한 최신 트렌드를 한발 앞서서 받아들이는 웹 개발자가 되고자 합니다.
